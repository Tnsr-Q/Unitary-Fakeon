# symbolic/hyperint

HyperInt configurations for the Fakeon master-integral programme.

| File                            | Scope                                  |
|---------------------------------|----------------------------------------|
| `crossedbox_PV.maple`           | Massless crossed-box, PV projection    |
| `crossedbox_massive_PV.maple`   | Massive crossed-box, 6-master 6×6 DE   |

Both scripts assume the residue matrices and boundary vector are loaded
from a companion `.m` data file generated by the Lean source of truth in
`Fakeon/Algebra/MassiveDE.lean`.
